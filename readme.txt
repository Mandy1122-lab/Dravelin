# Dravelin
這是一個整合戲劇與電影作品拍攝景點的平臺，旨在讓使用者輕鬆查詢戲劇和電影的拍攝景點，並探索各景點背後的拍攝作品。

## 功能特色
－ 景點查詢：根據戲劇或電影名稱，搜尋拍攝景點位置。
－ 作品探索：查看特定地點有哪些戲劇或電影曾在此拍攝。
－ 地圖整合：在互動地圖上可以一覽所有拍攝景點。
－ 活動專區：可以查看目前有協辦或是推廣的相關戲劇、電影活動

## 使用技術
－ 前端：HTML5, CSS3, JavaScript 
－ 後端：PHP
－ 資料庫：MySQL
－ 爬蟲：Python
－ 其他：Figma（設計原型）
